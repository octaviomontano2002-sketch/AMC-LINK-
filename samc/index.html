SOBRE EL PROYECTO

SISTEMA DE ASISTENCIA MAN CÉSPED (SAAM©)

GESTIÓN 2026

Este sistema ha sido desarrollado específicamente para atender las necesidades de registro de asistencia y rendimiento de la Academia de Música Man Césped. Integra herramientas de sincronización en tiempo real en la nube empleando Firebase Realtime Database, diseñado especialmente para optimizar la labor docente y administrativa.

INFORMACIÓN DEL SISTEMA

Desarrollador y Gestión: Dirección Académica / Sistemas AMC

Propiedad Institucional: Academia de Música Man Césped

Versión del Software: 1.4 (Build 2026)

Tecnologías Aplicadas: HTML5, CSS3, ES6 JavaScript puro, Firebase, y PDF Generator (jsPDF).
